Turkish Airlines is the official airline of “10th European Conference on Software Architecture (ECSA 2016)” and special discounts are offered on certain booking classes. In order to proceed with the online booking tool for Turkish Conventions please visit the Turkish Conventions website and use the event code “170TKM16” under delegate section.

Event code will be posted here soon and you will use that event code under delegate section to buy your flight ticket.

The European Conference on Software Architecture (ECSA) is the premier European software architecture conference, providing researchers, practitioners, and educators with a platform to present and discuss the most recent, innovative and significant findings and experiences in the field of software architecture research and practice. The conference will feature a research track, an industrial track, keynotes, workshops, tutorials, tool demonstrations, and panels.


ECSA 2016 will be held in Istanbul, Turkey (Istanbul Kültür University, Atakoy Campus)

This web page contains all the information about the conference, and all supporting activities. Please check this page regularly for updated information.
